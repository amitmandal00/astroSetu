# RUNBOOK

## Install
```
cd astrosetu
npm install --legacy-peer-deps --no-audit --no-fund
```

## Type Check
```
npm run type-check
```

## Build
```
npm run build
```

## Unit Tests
```
npm run test:unit
```

## Integration Tests
```
npm run test:integration
```

## E2E Tests
```
npm run test:e2e
```

## Release Gate
```
npm run release:gate
```
